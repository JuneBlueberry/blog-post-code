---
title: 7.以下typeof分别输出什么
date: 2021/05/07 11:19
categories: 
- 问答题
tags: 
- JavaScript
- 数据类型
---

```javascript
typeof 123            //numer
typeof '123'          //string
typeof true           //boolean
typeof null           //object
typeof undefined      //undefined
typeof Symbol()       //symbol
typeof []             //object
typeof {}             //object
typeof console        //object
typeof console.log    //function
typeof function(){}   //function
typeof new Date()     //object
```
